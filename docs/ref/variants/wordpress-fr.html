Variant pour WordPress

  • Examples pratiques
  • Questions ou commentaires?

Aperçu

Chef de project : Jobe Roberts (@Joberrr)

Objet

Adapte la BOEW pour l’utilisation avec WordPress.

Avantages

  • Facile à installer comme tout autre thème WordPress.
  • Prise en charge du français et de l’anglais dans une seule instance WordPress.
  • Conformes à WCAG 2.0 AA
  • Recours aux WAI-ARIA pour accroître l’accessibilité

Mise en œuvre

Comment mettre en œuvre les plugiciels requis

  1. qTranslate : Permet le soutien multilingue
    1. Ouvrir une session dans une instance WordPress à titre d’administrateur.
    2. Sélectionner « Languages » dans le menu « Settings ».
    3. Activer le soutien pour le français et désactiver le soutien pour les autres langues qui ne sont pas nécessaires.
  2. qTranslate meta : Permet d’étendre le soutien qTranslate pour la prise en charge des métadonnées
    1. Au moment de modifier une page ou un article, les onglets des langues française et anglaise seront disponibles lorsqu’on ajoute de nouveaux onglets de métadonnées.
    2. Les onglets de métadonnées de date sont pris en charge automatiquement par WordPress. Les autres onglets de métadonnées devront être réglés une seule fois pour l’ensemble du site. On peut effectuer cette opération en modifiant les lignes 25 à 30 dans header.php.
  3. Breadcrumb NavXT : Permet la prise en charge de la fonction de fils d’Ariane
    1. Sélectionner « Breadcrumb NavXT » dans le menu « Settings ».
    2. Sélectionner l'onglet « General ».
    3. Changer « blog » pouyr le nom du site Web sous « Main Site Breadcrumb: Main Site Home ».

Réglages facultatifs de WordPress

Settings - General

  1. Régler le titre en anglais et en français en ajoutant ce qui suit dans le champ « Site Title » : [:en]English Title[:fr]Titre en Français.
  2. Régler le fuseau horaire.

Settings - Discussion

  1. Sélectionner la case « An administrator must always approve the comment » pour l’option « Before a comment appears ».

Settings - Permalink

Le réglage par défaut utilise les numéros de page comme permaliens. Pour utiliser des adresses URL propres (c’est-à-dire des URL utilisant des mots), sélectionner « Day and Name » ou « Month and Name ». ISAPI Rewrite doit être installé si votre site tourne sous MS IIS.

Plugiciels facultatifs

  • Broken Link Checker
  • Google XML Sitemap generator
  • Simple Javascript Management (permet l’utilisation de différents scripts sur chaque page)
  • WPtouch (permet d’afficher un site sur un appareil mobile)
  • Search Unleashed ((améliore les fonctions de recherche)
  • Per Page Sidebars
  • Contact Form 7
