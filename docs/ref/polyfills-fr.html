Correctifs

Needs translation The polyfills are JavaScript components that mimic the behaviour of HTML5 tags in browsers that do not recognized such tags. They provide backwards compatibility for HTML5 in browsers that do not support HTML5.

  • Affichage des formules mathématiques ou scientifiques - Émule le MathML (anglais seulement) pour les navigateurs qui ne le supportent pas.
  • Barre coulissante - Émule l’élément <input type="range" /> pour les navigateurs qui ne le supportent pas.
  • Barre de progression - Émule l’élément <progress> pour les navigateurs qui ne le supportent pas.
  • Compteur (v3.1.0) - Émule l’élément <meter> pour les navigateurs qui ne le supportent pas.
  • Contenu affichable/masquable - Émule les éléments <details> et <summary> pour les navigateurs qui ne le supportent pas.
  • Mémoire de session (v3.1.0) - Émule l’objet Web Storage sessionStorage pour les navigateurs qui ne le supportent pas.
  • Mémoire persistante - Émule l’objet Web Storage localStorage pour les navigateurs qui ne le supportent pas.
  • Saisie automatique - Émule l'attribut list de l'élément <input>, ainsi que l'élément <datalist>, pour les navigateurs qui ne les supportent pas.
  • Sélecteur de date - Émule l'élément <input type="date" /> pour les navigateurs qui ne le supportent pas.
