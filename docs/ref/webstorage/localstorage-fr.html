Aperçu

Project lead: Paul Jackson (@pjackson28)

Objet

L'objet HTML5 localStorage est utilisé pour maintenir les données avec persistance (sans date d'expiration). Ce composant ajoute le soutien pour ces éléments dans les navigateurs qui n'ont pas déjà la prise en charge native.

Mise en œuvre

Utiliser l'objet localStorage pour maintenir les données avec persistance. Dans les cas où les navigateurs ne supportent pas cet objet, le polyfill est automatiquement chargé.

Code d'exemple

value = localStorage.getItem('variable');
localStorage.setItem('variable', value);

Développement

Maintenir les données avec persistance est dépendent sur le soutien natif pour l'objet localStorage et le correctif pour localStorage.

Le code pour le correctif pour localStorage se trouve dans la répertoire source de la BOEW :

  • js/polyfills/localstorage.js - contains the JavaScript code for the localStorage polyfill
