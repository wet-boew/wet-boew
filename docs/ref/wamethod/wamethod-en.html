Web accessibility assessment methodology

  • Working examples
  • Questions or comments?

Overview

Project Lead: Paul Jackson (@pjackson28)

Purpose

Provides an assessment methodology that assists with measuring conformance to the Web Content Accessibility Guidelines (WCAG) 2.0 Level A, AA, and AAA Success Criteria.

Recommended usage

  • Assessing conformance to the the Web Content Accessibility Guidelines (WCAG) 2.0 Level A, AA, and AAA Success Criteria.

Implementation

Migrate the *.html, *.css and *.js files in demos/wamethod to the Web site's layout and design.

Examples

  • English example (AA)
  • English example (AAA)
  • French example (AA)
  • French example (AAA)

Development

The code for the Web accessibility assessment methodlogy is located demos/wamethod:

  • demos/js/wamethod.js - contains the JavaScript code for the Web accessibility assessment methodology
  • demos/wamethod/css/wamethod.css - contains the CSS for the Web accessibility assessment methodology
  • demos/wamethod/css/wamethod-ie.css - contains the IE-specific CSS for the Web accessibility assessment methodology
