Méthodologie d’évaluation sur l’accessibilité des sites Web

  • Examples pratiques
  • Questions ou commentaires?

Aperçu

Responsable du projet : Paul Jackson (@pjackson28)

Objet

Offre une méthodologie d’évaluation qui permet de mesurer le degré de conformité aux Règles pour l’accessibilité des contenus Web (WCAG) 2.0 pour les critères de succès de niveau A, AA et AAA.

Utilisation recommandée

  • Mesurer le degré de conformité aux Règles pour l’accessibilité des contenus Web (WCAG) 2.0 pour les critères de succès de niveau A, AA et AAA.

Mise en œuvre

Migrer les dossiers *.html, *.css et *.js selon la présentation et le format du site Web.

Développement

Le code pour la méthodologie d’évaluation sur l’accessibilité des sites Web se trouve dans demos/wamethod :

  • js/workers/feedback.js - contient le JavaScript pour la méthodologie d’évaluation sur l’accessibilité des sites Web
  • js/sass/includes/_feedback.scss - contient le CSS pour la feuille de réponse
  • demos/js/wamethod.js - contient le JavaScript pour la méthodologie d’évaluation sur l’accessibilité des sites Web
  • demos/wamethod/css/wamethod.css - contient le CSS pour la méthodologie d’évaluation sur l’accessibilité des sites Web
  • demos/wamethod/css/wamethod-ie.css - contient le CSS spécifiue à IE pour la méthodologie d’évaluation sur l’accessibilité des sites Web

Enjeux connus
