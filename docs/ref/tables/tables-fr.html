Augmentation des tableaux

  • Exemples pratiques
  • Questions ou commentaires?

Aperçu

Responsable de projet : Jeremy Sivaneswaran (@jeresiv)

Objet

Ajoutes des fonctionnalités aux tableaux telles que la recherche, le tri, le filtrage et la pagination.

Avantages

  • Ajoutes la recherche, le tri, le filtrage et la pagination.
  • Conformes à WCAG 2.0 AA
  • Recours aux WAI-ARIA pour accroître l’accessibilité
  • Approche d'amélioration progressive

Mise en oeuvre

Consulter les exemples pratiques pour les étapes à suivre.

Exemples

  • Examples pratiques

Développement

L'augmentation des tableaux est dépendent sur DataTables (licence GPL v2, licence BSD).

Le code pour l'interface à onglets se trouve à plusieurs endroits dans le répertoire source de la BOEW :

  • js/workers/tables.js - contient le code JavaScript de l'augmentation des tableaux
  • js/sass/includes/_tables.scss - contient le CSS de l'augmentation des tableaux
  • js/sass/includes/_tables-ie.scss - contient le CSS spécifique à IE7/8 de l'augmentation des tableaux
  • js/images/tables/ - contient les images de l'augmentation des tableaux
  • js/dependencies/datatables.js - fichier script pour DataTables
  • js/dependencies/metadata.js - fichier script pour jQuery Metadata
