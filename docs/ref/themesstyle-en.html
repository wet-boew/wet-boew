Themes and style

Themes

  • Base theme (v3.1.0) - A theme template to aid in the creation of new themes.
  • GC Web Usability Theme - The GC Web usability theme was developed to comply with the Standard on Web Usability and is recommended for use with all Government of Canada websites.
  • GC Web Usability Intranet Theme - The GC Web usability intranet theme adapts the GC Web usability theme for intranet sites.
  • OGPL theme (v3.1.2) - The OGPL theme was developed to support the Open Government Platform (OGPL).
  • WET theme (v3.1.0) - The WET theme was developed to provide an example implementation of the Base theme.
  • CLF 2.0 Theme - This theme complies with CLF 2.0 while working with the Web Experience Toolkit.

Style

  • Design guide (v3.1.0) - Any website using a WET template should adhere to this guide.
