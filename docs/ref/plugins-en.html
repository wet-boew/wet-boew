Plugins

A WET plugin provides various types of functionality useful to a website and provided by calling the function in the class attribute of an HTML tag.

  • Archiving Content - Helps users identify Web pages that have been archived online.
  • Charts and graphs - Dynamically generates charts and graphs from table data.
  • Expand/collapse all (v3.1.0) - Creates buttons that expand and collapse all details elements on a page.
  • Events calendar - Dynamically generates a calendar interface for navigating a list of events.
  • Feedback form - Allows users to submit feedback for a specific Web page or Web site.
  • Footnotes - Provides a consistent, accessible way of handling footnotes across websites.
  • Form validation - Provides generic validation and error message handling for Web forms.
  • Geomap (v3.1.0) - Displays a dynamic map over which information from additional sources can be overlaid.
  • Language selector - Provides ASP, Javascript, Perl, and PHP approaches for automatically linking to different language versions of the current page.
  • Lightbox - Helps build a photo gallery on a web page.
  • Menu bar - Provides an interactive menu bar with optional mega menu sub-menus.
  • Multimedia player - Provides a multimedia player for embedding accessible video and audio into web pages.
  • Prettify - Provides syntax highlighting of source code snippets in an HTML page using google-code-prettify.
  • Responsive images (v3.1.0) - Provides support for responsive images by emulating the proposed HTML5 <picture> element.
  • Session timeout - Helps Web asset owners to provide session timeout and inactivity timeout functionality.
  • Share widget - Facilitates sharing Web content on social media platforms.
  • Slide-out tab - Provides a tab at the edge of the content area that can slide out to show content.
  • Tabbed interface - Dynamically transforms multiple sections of content into a tabbed interface.
  • Table enhancement (v3.1.0) - Integrates the DataTables plugin into WET providing searching, sorting, filtering, pagination and other advanced features for tables.
  • Text highlighting - Automatically highlights certain words on a Web page. The highlighted words can be selected via the query string.
  • Web feeds - Aggregates and displays entries from one or more Web feeds.
  • Zebra striping - Adds alternating highlights to rows in lists or tables.
