Column Group Impact - Table Usability - Web Experience Toolkit (WET)

Please note the column group (colgroup element) width should be equal of the table width as per his cell, spanned or not, for a any row.

You have seen some common example, but now let see how the use of the colgroup element can have a huge impact on how the data can be interpretated.

The following example show how the table parser and any widget that use table would see your data. In regards of the colgroup element, my research conduct me to use this methodology because it seem to be most common but I fully understand that it is possible to have a different interpretation.

To show you all the possibility, we will always use the same table format and the example above will only play on how the colgroup element is defined and the table caption

If you hover the fift column in the table 3 you will be noticed that there are no column header highlight. That is because the table parser when in a issue when it was trying to understand the tabular data. The table validator give the error 10 - The cell used to represent the data at level: 1 must encapsulate his group and be the same. So what that means ? Simply the table parser saw two distinct data group at different column data level because the superior cell header for the summary column can not be mixed between the data at level 1 and level 2. The fix is pretty simple, use three colgroup element. See the next table (table 4).

Explaination of the table 6. You will remark that their are no column cell header highlight at all. That is because the column group don't make sense. The HTML Table validator will throw two error now. The error 10 (see the explaination of the table 3) and the error 9 Error in you header row group, there are cell that are crossing more than one colgroup under the level:1. This means that the cell header used to represent the column grouping are not representative of the colgroup element defined with the table. To get it fixed you will need to either redefine the column grouping or change the cell heading structure.
