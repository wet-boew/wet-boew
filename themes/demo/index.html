Nested grids padded in or flush

By default, nested grids are flush left unless you add background-*, a border-* or a module-*. In those cases they pad left and right so the inner content is not touching the permimeter. To override this, use the padding-none CSS (or any other padding-* CSS to increase or decrease the default).
