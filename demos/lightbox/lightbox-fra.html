Light Box

Table des matières

  • Caractéristiques
  • Classes CSS
  • Exemples de photos seules
  • Exemples d'album photos simple
  • Exemples d'album photos caché

Caractéristiques

  • Affiche les photos dans l'outil Fancybox
  • Des classes CSS facile À utiliser pour ajouter le Lightbox À votre page.
  • Utilisation du clavier pour une navigation facile.

Classes CSS

Cette outil utilise les classes CSS suivantes :

Pour les Photos seules

  • lightbox
  • lightbox-item

Pour un album de photos

  • Lightbox-group
  • lightbox-group-item

Pour un album de photos cachés

  • lightbox-group
  • lightbox-group-item-hidden
  • hidden

Examples

Photos seules (sans album)

Photos

Codage

			
<div class="wet-boew-lightbox">
  <div class="lightbox">
    <p><a class="lightbox-item" href="images/your_image.jpg" title="Title Image">
      <img src="images/image_source.jpg" alt="Texte alternatif pour les photos" class="image-actual" />
    </a></p>
    ...	
  </div>
</div>
