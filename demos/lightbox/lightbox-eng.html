Light Box

Table of contents

  • Features

  • Examples of Single Pictures
  • Examples of Picture Albums

Features

  • Display images in Fancybox plug-in.
  • Easy to use CSS classes to implement the Lightbox in your page.
  • Easy navigation in photo galleries using keyboard.

CSS Classes

This plug-in uses the following CSS Classes.

For single pictures

  • lightbox
  • lightbox-item

For Picture Albums

  • Lightbox-group
  • lightbox-group-item

For Hidden Picture Albums

  • lightbox-group
  • lightbox-group-item-hidden
  • hidden
