Responsive Images

Table of contents

Features

This plugin provides support for responsive images which allows you to specify multiple sources for an image that are optimized for different resolutions and devices.

CSS media queries are used to determine which image source should be loaded. As a result responsive images will only work in modern browers that support CSS media queries.

Usage

Add the wet-boew-responsiveimg CSS class to any element on your page. This will trigger the plugin, which in turn uses the Picturefill library to create the responsive image(s).

Data Attributes
Attribute Purpose
data-picture Identifies the <div> as a responsive image.
data-alt Alternate text that will be used for the image's alt attribute.
data-src Path to the image.
data-media CSS media query that determines when the image source is used.

Examples

Browser Width

This example uses min-width to determine which image should be displayed. Resize the browser window to see the different images load.

Photo by Noema Pérez, licensed under Creative Commons.

View code 
<div data-picture data-alt="Image alt text" class="wet-boew-responsiveimg">	
  <!-- Image sources for different minimum browser widths -->
  <div data-src="images/small.jpg"></div>
  <div data-src="images/medium.jpg" data-media="(min-width: 500px)"></div>
  <div data-src="images/large.jpg" data-media="(min-width: 960px)"></div>
  <div data-src="images/extralarge.jpg" data-media="(min-width: 1200px)"></div>

  <!-- Default image for IE8 and below (no CSS media query support) -->
  <!--[if (lt IE 9) & (!IEMobile)]>        	
    <div data-src="images/parliament-hill-large.jpg"></div>	
  <![endif]-->	

  <!-- Fallback image for non-JS browsers. -->
  <noscript><img src="images/large.jpg" alt="Image alt text"></noscript>
</div>

High Pixel Density

This example uses two media queries to serve high resolution images to devices that will benefit from them (i.e. Retina displays).

Photo by snowpeak, licensed under Creative Commons.

View code 
<div data-picture data-alt="False Kiva, located in the Island in the Sky district of Canyonlands">
  <!-- Image sources for different device pixel ratios -->
  <div data-src="images/cave-low-res.jpg"></div>
  <div data-src="images/cave-high-res.jpg" data-media="(-webkit-min-device-pixel-ratio: 2),(min-resolution: 192dpi)"></div>

  <!-- Default image for IE8 and below (no CSS media query support) -->
  <!--[if (lt IE 9) & (!IEMobile)]>        	
    <div data-src="images/cave-low-res.jpg"></div>	
  <![endif]-->	

  <!-- Fallback content for non-JS browsers -->
  <noscript><img src="images/cave-low-res.jpg" alt="False Kiva, located in the Island in the Sky district of Canyonlands"></noscript>
</div>
