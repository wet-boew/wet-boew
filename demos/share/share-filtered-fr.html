Gadget de partage - Sites filtrés

  • Documentation
  • Questions ou commentaires?

Les sites pour partager cette page Web sont filtrés avec le code suivant dans settings.js :

var wet_boew_share = {
   sites : ['del.icio.us', 'digg', 'facebook', 'google', 'linkedin', 'reddit', 'stumbleupon', 'twitter', 'yahoobuzz']
};

Les sites pourraient être filtrés aussi de page en page en utilisant l'attribut data-wet-boew :

<div class="wet-boew-share" data-wet-boew="{sites:['digg','facebook','twitter']}"></div>
