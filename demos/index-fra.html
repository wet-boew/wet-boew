Exemples pratiques

Nom Catégorie Description Mots clés
Emulateur pour la conception Web réactive Autre Affiche une page Web du thème de la BOEW aux largeurs de fenêtre d'affichage et aux tailles d'appareil courantes.
Méthodologie d’évaluation sur l’accessibilité des sites Web Autre Offre une méthodologie d’évaluation qui permet de mesurer le degré de conformité aux Règles pour l’accessibilité des contenus Web (WCAG) 2.0 pour les critères de succès de niveau A, AA et AAA.
Outils de développement pour le contenu optimal (v3.1.0) Autre Des outils aidant à la création d’éléments de pages Web. Inclus un outil de validation de tableau).
Répartition des responsabilités en accessibilité Autre Une répartition des responsabilités associées aux critères de succès pour WCAG 2.0 sur l’ensemble de la chaîne de production Web.
Système de grilles Cadre fondamental Une grille de mise en page, des modules et autres fonctionnalités servant à disposer les éléments d'une page de manière visuellement cohérente.
Affichage des formules mathématiques ou scientifiques Correctifs Émule le MathML (en anglais seulement) pour les navigateurs qui ne le supportent pas.
Barre coulissante Correctifs Émule l’élément <input type="range" /> pour les navigateurs qui ne le supportent pas.
Barre de progression Correctifs Émule l’élément <progress> pour les navigateurs qui ne le supportent pas.
Compteur (v3.1.0) Correctifs Émule l’élément <meter> pour les navigateurs qui ne le supportent pas.
Contenu affichable/masquable Correctifs Émule les éléments <details> et <summary> pour les navigateurs qui ne le supportent pas.
Mémoire de session (v3.1.0) Correctifs Émule l’objet Web Storage sessionStorage pour les navigateurs qui ne le supportent pas. HTML5
Mémoire persistante Correctifs Émule l’objet Web Storage localStorage pour les navigateurs qui ne le supportent pas. HTML5
Saisie automatique Correctifs Émule l'attribut list de l'élément <input>, ainsi que l'élément <datalist>, pour les navigateurs qui ne les supportent pas.
Sélecteur de date Correctifs Émule l'élément <input type="date" /> pour les navigateurs qui ne le supportent pas.
Augmentation des tableaux (v3.1.0) Plugiciels Ajoutes des fonctionnalités aux tableaux telles que la recherche, le tri, le filtrage et la pagination.
Barre de menu Plugiciels Une barre de menu interactive comprenant des sous-menus optionnels pour le mégamenu.
Calendrier d'événements Plugiciels Un calendrier servant à consulter une liste d'événements.
Contenu archivé Plugiciels Ajoute un avis d'archivage à une page Web.
Expiration de la session Plugiciels Ferme la session de l'utilisateur après un certain délai.
Fils de syndication Plugiciels Affiche les articles d’un ou plus fils de syndication.
Formulaire de rétroaction Plugiciels Un formulaire normalisé pour soumettre une rétroaction au sujet d'une page ou d'un site Web particulier.
Gadget de partage Plugiciels Permet à l'utilisateur de partager un contenu Web sur les plateformes de médias sociaux.
Géocarte (v3.1.0) Plugiciels Affiche une carte dynamique sur laquelle des informations de sources supplémentaires peuvent être superposées.
Graphiques Plugiciels Génère des graphiques à partir de données d’un tableau.
Images réactives (v3.1.0) Plugiciels Émule l’élément proposé <picture> pour permettre que la source d'une image soit déterminée en fonction de Media Queries CSS.
Interface à onglets Plugiciels Comprime plusieurs sections de contenu dans le même espace physique et fourni des onglets qui permettent à l'utilisateur de choisi la section de contenu à afficher.
Lecteur multimédia Plugiciels Un lecteur multimédia accessible permettant d'intégrer des contenus vidéo et audio sur les pages Web.
Lightbox Plugiciels Crée une galerie photo sur une page Web.
Mise en surbrillance de text Plugiciels Met en surbrillance sur une page Web les mots choisis.
Notes de bas de page Plugiciels Une méthode accessible pour afficher des notes de bas de page.
Onglet coulissant Plugiciels Masque une section de contenu et affiche un onglet à l'extrémité de la zone de contenu permettant de faire basculer l'état d'affichage de ce contenu.
Prettify Plugiciels Affiche de la syntaxe des extraits de code source.
Rayage du zèbre Plugiciels Ajoute des rayures aux listes et aux tableaux.
Sélecteur de langue Plugiciels Automatise la création de liens vers les versions de la page actuelle dans différentes langues.
Tout afficher/masquer (v3.1.0) Plugiciels Permet d'afficher ou de masquer tous les éléments <details> d'une page.
Validation de formulaires Plugiciels Effectue la validation de formulaires Web selon un ensemble de règles de base avant qu'ils soient soumis.
Exemples de contenus optimaux Thèmes et style Des exemples et des explications de la façon de coder les divers éléments d’une page Web afin de s'assurer qu'ils soient accessibles.
Guide de conception (v3.1.0) Thèmes et style Une description de l'utilisation prévue des éléments CSS et des scripts de la Boîte à outils de l'expérience Web.
Thème de base (v3.1.0) Thèmes et style Un modèle de thème pour la création de nouveaux thèmes.
Thème de la Boîte à outils de l'expérience Web (BOEW) (v3.1.0) Thèmes et style Un exemple de thème conçu à partir du thème de base.
Thème de la facilité d’emploi Web GC Thèmes et style Un thème qui met en place la mise en page et la mise en forme exigés par la Norme sur la facilité des sites Web. Il est recommandé de se servir de ce thème sur les sites Web du gouvernement du Canada.
Thème de la facilité d’emploi Web GC pour les sites intranet Thèmes et style Une version du thème de la facilité d'emploi des sites Web du GC pour les sites intranet.
Thème de la Normalisation des sites Internet 2.0 Thèmes et style Un thème qui met en place la mise en page et la mise en forme exigées par la Normalisation des sites Internet 2.0.
Thème de la Plate-forme de gouvernement ouvert (PGO) (v3.1.2) Thèmes et style Un thème qui met en place la mise en page et la mise en forme pour la Plate-forme de gouvernement ouvert (PGO).
Variant pour Drupal Variants Adapte la BOEW pour l’utilisation avec Drupal.
Variant pour Jekyll (v3.1.0) Variants Adapte la BOEW pour l’utilisation avec Jekyll.
Variant pour PHP Variants Adapte la BOEW pour l’utilisation avec PHP.
Variant pour SSI Variants Adapte la BOEW pour l’utilisation avec SSI.
Variant pour WordPress Variants Adapte la BOEW pour l’utilisation avec WordPress.
