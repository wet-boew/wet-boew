Exemples pratiques

Nom Catégorie Description Mots clés
Emulateur pour la conception Web réactive Autre Affiche une page Web du thème de la BOEW aux largeurs de fenêtre d'affichage et aux tailles d'appareil courantes.
Méthodologie d’évaluation sur l’accessibilité des sites Web Autre Offre une méthodologie d’évaluation qui permet de mesurer le degré de conformité aux Règles pour l’accessibilité des contenus Web (WCAG) 2.0 pour les critères de succès de niveau A, AA et AAA.
Outils de développement pour le contenu optimal (v3.1.0) Autre Des outils aidant à la création d’éléments de pages Web. Inclus un outil de validation de tableau).
Répartition des responsabilités en accessibilité Autre Une répartition des responsabilités associées aux critères de succès pour WCAG 2.0 sur l’ensemble de la chaîne de production Web.
Système de grilles Cadre fondamental Une grille de mise en page, des modules et autres fonctionnalités servant à disposer les éléments d'une page de manière visuellement cohérente.
Affichage des formules mathématiques ou scientifiques Correctif Émule le MathML (en anglais seulement) pour les navigateurs qui ne le supportent pas.
Barre coulissante Correctif Émule l’élément <input type="range" /> pour les navigateurs qui ne le supportent pas.
Barre de progression Correctif Émule l’élément <progress> pour les navigateurs qui ne le supportent pas.
Compteur (v3.1.0) Correctif Émule l’élément <meter> pour les navigateurs qui ne le supportent pas.
Contenu affichable/masquable Correctif Émule les éléments <details> et <summary> pour les navigateurs qui ne le supportent pas.
Mémoire de session (v3.1.0) Correctif Émule l’objet Web Storage sessionStorage pour les navigateurs qui ne le supportent pas. HTML5
Mémoire persistante Correctif Émule l’objet Web Storage localStorage pour les navigateurs qui ne le supportent pas. HTML5
Saisie automatique Correctif Émule l'attribut list de l'élément <input>, ainsi que l'élément <datalist>, pour les navigateurs qui ne les supportent pas.
Sélecteur de date Correctif Émule l'élément <input type="date" /> pour les navigateurs qui ne le supportent pas.
Augmentation des tableaux (v3.1.0) Plugiciel Ajoutes des fonctionnalités aux tableaux telles que la recherche, le tri, le filtrage et la pagination.
Barre de menu Plugiciel Une barre de menu interactive comprenant des sous-menus optionnels pour le mégamenu.
Calendrier d'événements Plugiciel Un calendrier servant à consulter une liste d'événements.
Contenu archivé Plugiciel Ajoute un avis d'archivage à une page Web.
Expiration de la session Plugiciel Ferme la session de l'utilisateur après un certain délai.
Fils de syndication Plugiciel Affiche les articles d’un ou plus fils de syndication.
Formulaire de rétroaction Plugiciel Un formulaire normalisé pour soumettre une rétroaction au sujet d'une page ou d'un site Web particulier.
Gadget de partage Plugiciel Permet à l'utilisateur de partager un contenu Web sur les plateformes de médias sociaux.
Géocarte (v3.1.0) Plugiciel Affiche une carte dynamique sur laquelle des informations de sources supplémentaires peuvent être superposées.
Graphiques Plugiciel Génère des graphiques à partir de données d’un tableau.
Images réactives (v3.1.0) Plugiciel Émule l’élément proposé <picture> (en anglais seulement) pour permettre que la source d'une image soit déterminée en fonction de Media Queries CSS.
Interface à onglets Plugiciel Comprime plusieurs sections de contenu dans le même espace physique et fourni des onglets qui permettent à l'utilisateur de choisi la section de contenu à afficher.
Lecteur multimédia Plugiciel Un lecteur multimédia accessible permettant d'intégrer des contenus vidéo et audio sur les pages Web.
Lightbox Plugiciel Crée une galerie photo sur une page Web.
Mise en surbrillance de text Plugiciel Met en surbrillance sur une page Web les mots choisis.
Notes de bas de page Plugiciel Une méthode accessible pour afficher des notes de bas de page.
Onglet coulissant Plugiciel Masque une section de contenu et affiche un onglet à l'extrémité de la zone de contenu permettant de faire basculer l'état d'affichage de ce contenu.
Prettify Plugiciel Affiche de la syntaxe des extraits de code source.
Rayage du zèbre Plugiciel Ajoute des rayures aux listes et aux tableaux.
Sélecteur de langue Plugiciel Automatise la création de liens vers les versions de la page actuelle dans différentes langues.
Tout afficher/masquer (v3.1.0) Plugiciel Permet d'afficher ou de masquer tous les éléments <details> d'une page.
Validation de formulaires Plugiciel Effectue la validation de formulaires Web selon un ensemble de règles de base avant qu'ils soient soumis.
Exemples de contenus optimaux Thème et style Des exemples et des explications de la façon de coder les divers éléments d’une page Web afin de s'assurer qu'ils soient accessibles.
Guide de conception (v3.1.0) Thème et style Une description de l'utilisation prévue des éléments CSS et des scripts de la Boîte à outils de l'expérience Web.
Thème de base (v3.1.0) Thème et style Un modèle de thème pour la création de nouveaux thèmes.
Thème de la Boîte à outils de l'expérience Web (BOEW) (v3.1.0) Thème et style Un exemple de thème conçu à partir du thème de base.
Thème de la facilité d’emploi Web GC Thème et style Un thème qui met en place la mise en page et la mise en forme exigés par la Norme sur la facilité des sites Web. Il est recommandé de se servir de ce thème sur les sites Web du gouvernement du Canada.
Thème de la facilité d’emploi Web GC pour les sites intranet Thème et style Une version du thème de la facilité d'emploi des sites Web du GC pour les sites intranet.
Thème de la Normalisation des sites Internet 2.0 Thème et style Un thème qui met en place la mise en page et la mise en forme exigées par la Normalisation des sites Internet 2.0.
Thème de la Plate-forme de gouvernement ouvert (PGO) (v3.1.2) Thème et style Un thème qui met en place la mise en page et la mise en forme pour la Plate-forme de gouvernement ouvert (PGO).
Variant pour Drupal Variant Adapte la BOEW pour l’utilisation avec Drupal.
Variant pour Jekyll (v3.1.0) Variant Adapte la BOEW pour l’utilisation avec Jekyll.
Variant pour PHP Variant Adapte la BOEW pour l’utilisation avec PHP.
Variant pour SSI Variant Adapte la BOEW pour l’utilisation avec SSI.
Variant pour WordPress Variant Adapte la BOEW pour l’utilisation avec WordPress.
