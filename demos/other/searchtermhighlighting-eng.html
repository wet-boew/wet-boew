Search Term Highlighting Showcase - GC Web Usability theme

The search term highlighting sub project consists of a search box that allows you to enter the keywords your looking for. On keypress, the plugin will start searching through the text of an element and highlight all matches found that are longer than the minimum length specified.

Benefits

  • Helps users to find certain words on a Web page
  • Includes a simple search form that allow users to set which words to highlight
  • Conforms to WCAG 2.0 AA
  • Uses WAI-ARIA to enhance accessibility
  • Progressive enhancement approach
  • Supports Firefox, Opera, Safari, Chrome, and IE 7+
