Labels - Charts and graphs

  • Documentation
  • Questions or comments?

Requirement

WET v3.1 or later

Position

This is how you specify which row or column will be used to label the charts. The column is only available when the table is reversed.

wb-charts-labelposition-x où x représente un nombre positif associé à la rangée ou colonne dans la table. La première rangée ou colonne est représenté par le chiffre 1. La valeur par défaut est la rangée ou colonne le plus profond dans la groupe d'en-têtes.

Exemples

First row used to label the chart - wb-charts-labelposition-1

Uniform Tick options

By default, the tick is uniform because each table slot would have the same value (uniform value between slots). When deactivated, only the highest header row or column would get the uniform value. Child cells would get devised value attribution.
