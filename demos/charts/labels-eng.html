Labels - Charts and graphs - Web Experience Toolkit (WET)

  • Documentation
  • Questions or comments?

Requirement

WxT 3.1 or higher

Position

This is how you can specify wich row or column that would be use to label the charts. The column is only available when the table is reversed.

wb-charts-labelposition-x Where x represent a positive number associated to the row/col position of the table. The first row/col is represented by the number 1. The default value is the deppest row/col used in the heading group.

Examples

First row use to label the chart - wb-charts-labelposition-1

Uniform Tick options

By default, the tick is called uniform because each table slot would have the same value (Uniform value between slots). When deactivated, only the highest header row/col would get uniform value. Childs cells would get divised value attribution.

Examples
